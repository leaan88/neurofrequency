# Neurofrequency
Genera música binaural en 432 Hz y solfegios basada en principios taoístas y ondas cerebrales simuladas. Diseñado para integrarse con interfaces cerebro-máquina como Neuralink.

## Características
- Selección de nota raíz, escala (mayor, menor, dorian, etc.) y elemento taoísta (agua, fuego, madera, tierra, metal).
- Ajuste de balance Yin/Yang para personalizar armonías y melodías.
- Generación de melodías MIDI exportables para Ableton Live.
- Frecuencias binaurales (4-20 Hz) en el rango de ondas cerebrales (delta, theta, alfa) para efectos sanadores.
- Interfaz React con diseño intuitivo, colores dinámicos y feedback visual.

## Frecuencias y sus Beneficios

### 432 Hz - Frecuencia Universal
La frecuencia de **432 Hz** es conocida como la "frecuencia universal" o "frecuencia cósmica". Se cree que está en armonía con las ondas cerebrales naturales y el ritmo del universo.

**Beneficios:**
- Promueve la calma y la relajación profunda
- Reduce la ansiedad y el estrés
- Mejora la concentración y la claridad mental
- Facilita la meditación y el estado de consciencia expandida
- Se asocia con la sanación vibracional a nivel celular

**Cómo usar:** Usa esta frecuencia durante meditaciones largas, sesiones de yoga o antes de dormir para un descanso profundo.

## Cómo Usar Correctamente

### Para Meditación
1. Busca un lugar tranquilo y cómodo
2. Ponte auriculares estéreo de buena calidad (importante para binaural)
3. Relájate y respira profundamente
4. Escucha durante 15-30 minutos mínimo para mejores resultados
5. Mantén una intención clara o afirmación positiva

### Para Sanación
1. Selecciona la frecuencia que corresponde a tu necesidad
2. Escucha en volumen moderado (no demasiado fuerte)
3. Practica regularmente (diariamente si es posible)
4. Combina con visualización o afirmaciones
5. Permítete sentir cualquier emoción que surja durante el proceso

### Recomendaciones Generales
- **Consistencia:** Los beneficios se acumulan con escuchas regulares
- **Duración:** Sesiones de 20-40 minutos son ideales
- **Horario:** Por las mañanas para energía, por las noches para relajación
- **Entorno:** Evita distracciones y crea un espacio sagrado
- **Intención:** Establece claramente qué deseas sanar o transformar
